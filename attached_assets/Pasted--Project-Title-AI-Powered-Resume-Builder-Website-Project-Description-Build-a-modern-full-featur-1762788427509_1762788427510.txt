# Project Title
AI-Powered Resume Builder Website

# Project Description
Build a modern, full-featured **AI Resume Builder Website** that allows users to create, edit, and download professional resumes with smart AI assistance. The platform should support real-time editing, customizable templates, resume analytics, and AI-powered suggestions. It should also include user authentication, autosave, resume sharing, and premium subscription tiers.

The website should be fast, responsive, user-friendly, and optimized for mobile and desktop devices.

# Technical Requirements

## Frontend
- **React + Tailwind CSS** (modern, responsive design)
- State management: **Zustand or Redux Toolkit**
- Routing: React Router
- **Rich text editing** for description fields
- PDF rendering: `jspdf`, `react-pdf`, or server-side export
- API communication via Axios or Fetch
- Component-driven UI for modularity

## Backend
- **Node.js + Express.js** REST API
- **MongoDB or PostgreSQL** for data storage
- JWT Authentication
- Cloud storage (AWS S3 or Firebase) for resume templates and files
- Autosave endpoint for real-time updates
- AI endpoints for GPT-style integration

## AI Integration
- Integrate **OpenAI API** (GPT models) or **Gemini API**
- Use AI for:
  - Summary writing
  - Bullet point generation
  - Job keyword analysis
  - Resume scoring
  - Cover letter generation
- AI caching and request throttling for performance optimization

## Infrastructure
- **Vercel / Netlify** for frontend deployment
- **Render / Railway / AWS Lambda** for backend
- SSL & SEO optimization
- Web analytics integration (Google Analytics, PostHog)

# User Flow Summary
1. User signs up or logs in  
2. Chooses a resume template  
3. Fills sections with guided editor  
4. Gets live preview of changes  
5. Optionally uses AI to generate or enhance content  
6. Downloads PDF or shares public link  
7. (Optional) Upgrades to premium for advanced templates or AI features
